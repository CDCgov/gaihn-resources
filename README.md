# About Global Action in Healthcare Network

What to know
- The Global Action in Healthcare Network (GAIHN) is a global collaborative network combating emerging infectious disease threats in healthcare settings through rapid detection, prevention and response.
- GAIHN members include countries, institutions and partners at global, regional, national and subnational levels.
- GAIHN was launched in 2021 as part of CDC's Global Antimicrobial Resistance Laboratory and Response Network.
- Members work together to address healthcare-associated infections and antimicrobial resistance in healthcare settings.
